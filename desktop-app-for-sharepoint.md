# <a name="desktop-app-for-sharepoint"></a>SharePoint için Masaüstü uygulama

Kuruluşunuzun özel Önizleme programı için masaüstü uygulaması için SharePoint girilir. Microsoft için SharePoint (özel Önizleme) masaüstü uygulaması şu anda desteklemiyor. Lütfen kayıtlı Microsoft Özel önizleme ortaklarınız ile masaüstü uygulaması için SharePoint Yardım almak için kullanın.

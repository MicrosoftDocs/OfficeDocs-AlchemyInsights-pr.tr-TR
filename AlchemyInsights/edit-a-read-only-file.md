# <a name="edit-a-read-only-file"></a>Salt okunur bir dosyayı düzenleme

Belgenin adını tıklattığınızda, belge kitaplıklarında, Office programı beklenmedik şekilde salt okunur dosya olarak açmak. Düzenlemek üzere açmak için (üç nokta) **Aç** ' ı tıklatın ve sonra **Düzenle** ' yi tıklatın.

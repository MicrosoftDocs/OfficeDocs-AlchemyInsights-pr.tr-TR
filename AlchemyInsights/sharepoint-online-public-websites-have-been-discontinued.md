# <a name="sharepoint-online-public-websites-have-been-discontinued"></a>SharePoint Online genel web siteleri durduruldu

SharePoint Online genel web siteleri durduruldu.

Bir SharePoint Online genel web sitesi oluşturduysanız, 31 Mart 2018'e kadar kullanmaya devam edebilirsiniz. Herkese açık bir web sitesine sahip olmaya devam etmek istiyorsanız, yeni bir sağlayıcıyla yeni bir web sitesi oluşturmanız gerekir. SharePoint Online genel web sitenizdeki içerik silindi.

Genel web sitesinin durdurulması hakkında daha fazla bilgi için, [özellik durdurma işlemi yle ilgili ayrıntılara](https://go.microsoft.com/fwlink/?linkid=866980)bakın.

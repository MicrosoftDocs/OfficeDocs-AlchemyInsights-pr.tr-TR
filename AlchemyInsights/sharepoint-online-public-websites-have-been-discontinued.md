# <a name="sharepoint-online-public-websites-have-been-discontinued"></a>Genel SharePoint çevrimiçi Web siteleri yarıda kesildi

Genel SharePoint çevrimiçi Web siteleri yarıda kesildi.

Genel SharePoint çevrimiçi bir Web sitesi oluşturduysanız, 31 Mart 2018 kullanmaya devam edebilirsiniz. Ortak bir Web sitesine sahip olmak devam etmek istiyorsanız, yeni bir sağlayıcı ile yeni bir tane oluşturmanız gerekir. Ortak Web sitenizde SharePoint çevrimiçi içeriği silindi.

[Özellik discontinuation işleminin ayrıntılarını](https://go.microsoft.com/fwlink/?linkid=866980)ortak Web sitesine discontinuation hakkında daha fazla bilgi için bkz.

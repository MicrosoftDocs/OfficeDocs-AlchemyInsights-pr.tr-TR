# <a name="learn-about-external-calendar-sharing-options"></a>Dış takvim paylaşım seçenekleri hakkında bilgi edinin

Kuruluşunuz dışındaki kişilerle toplantı zamanlamalısınız, Microsoft 365 takvimleri "harici kullanıcılar" için kullanılabilir hale getirmenize olanak tanır, serbest/yoğun zamanı görmesi gereken ancak Microsoft 365 ortamınız için kullanıcı hesabı olmayanlar. Daha fazla bilgi için [bkz.](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users)

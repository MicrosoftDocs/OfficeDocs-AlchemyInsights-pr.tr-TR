# <a name="restore-deleted-users-and-groups"></a>Silinen kullanıcıları ve grupları geri yükleme

Bir kullanıcıyı yanlışlıkla silinen bir kullanıcıyı geri yüklemek için, [Azure Active Directory kullanarak yeni silinen kullanıcıyı geri yükleme veya kaldırma](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-users-restore?context=azure/active-directory/users-groups-roles/context/ugr-context)konusuna bakın.

Silinen grubu geri yüklemek için, [Azure Active Directory 'de silinmiş bir Microsoft 365 grubunu geri yükleme](https://docs.microsoft.com/azure/active-directory/users-groups-roles/groups-restore-deleted)bölümüne bakın.

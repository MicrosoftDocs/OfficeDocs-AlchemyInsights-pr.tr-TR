# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 erişim reddedildi, kötüye kullanım algılandı

Normalde bu ileti, kullanıcılar, Microsoft 365 'daki yeni kiralara atanmış olan ilk *. onmicrosoft.com* etki ALANıNı kullanarak IP adreslerinden e-posta iletileri göndermesinden kaynaklanır. Bu sorunu çözmenin en kolay yolu şudur:

1. [Kiracınıza etki alanı ekleyin](https://docs.microsoft.com/microsoft-365/admin/setup/add-domain).

2. [Kullanıcılarınızın birincil e-posta adresini](https://docs.microsoft.com/microsoft-365/admin/add-users/change-a-user-name-and-email-address) yeni eklediğiniz özel etki alanıyla değiştirin.

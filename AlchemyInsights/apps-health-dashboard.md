# <a name="microsoft-365-apps-health-dashboard-in-the-microsoft-365-apps-admin-center"></a>Microsoft 365 Uygulamaları yönetim merkezinde Microsoft 365 Uygulamaları durum panosu

Microsoft 365 Uygulamaları yönetim merkezinde bulunan Microsoft 365 Uygulamaları sistem durumu panosu, istemci cihazlarında performanslarının görünürlüğünü sağlayarak, sorunları iyileştirmeye ve gidermeye yardımcı olacak kılavuzlar sunarak Office uygulamalarınızı güncel tutmanıza yardımcı olur. Daha fazla bilgi için [bkz. Microsoft Uygulamaları yönetim merkezine ve](https://docs.microsoft.com/deployoffice/admincenter/overview) Microsoft [365 Uygulamaları durumuna genel bakış.](https://docs.microsoft.com/deployoffice/admincenter/microsoft-365-apps-health)

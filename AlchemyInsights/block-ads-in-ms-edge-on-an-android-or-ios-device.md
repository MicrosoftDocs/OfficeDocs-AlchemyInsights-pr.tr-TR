# <a name="block-ads-in-microsoft-edge-on-an-adroid-or-ios-device"></a>Microsoft Edge 'de bir Adroid veya iOS aygıtında reklamları engelleme

AdBlock Plus tarafından kabul edilen kabul edilen tüm reklamları engellemek için:
1. Select **...** > **Ayarları**  >  **İçerik engelleyicileri**.
2. Reklamları engelle 'yi açın.

Tüm reklamları engellemek için:
1. Select **...** > **Ayarları**  >  **İçerik engelleyicileri**.
2. Reklamları engelle 'yi açın.
3. **Gelişmiş ayarlar**'ı seçin.
4. Kabul edilebilir reklamları kabul edin.

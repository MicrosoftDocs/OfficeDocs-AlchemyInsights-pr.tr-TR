# <a name="unknown-user-appears-in-teams-meetings-or-chats"></a>Ekip toplantıları veya sohbetlerde "Bilinmeyen Kullanıcı" görüntüleniyor

Kullanıcı Microsoft 365 uygulamasından silinmiş olabilir. Bu durumda, kullanıcıyı ekiplerde silmeyi deneyin.  

Takımlar kapatıldığında aşağıdaki Önbelleği temizlemeyi deneyin, ardından ekipleri yeniden Başlat: `C:\Users\<USER>\AppData\Roaming\Microsoft\Teams`

Bu sorunla ilgili güncelleştirmeler için,  [kaldırılan Kullanıcı Microsoft ekiplerde "Bilinmeyen Kullanıcı" olarak görüntülenir](https://docs.microsoft.com/MicrosoftTeams/troubleshoot/known-issues/removed-user-appears-as-unknown).

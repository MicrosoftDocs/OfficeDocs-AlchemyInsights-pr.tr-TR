# <a name="delete-a-sharepoint-site-collection"></a>SharePoint site koleksiyonunu sil

1. Klasik SharePoint Yönetim Merkezi, site koleksiyonları sayfasında, site koleksiyonu yanındaki onay kutusunu seçin.
    
2. Site koleksiyonlarını sekmesinde, **Sil**' i seçin.

# <a name="add-or-remove-a-custom-role"></a>Özel rol ekleme veya kaldırma

Azure kaynaklarını yönetmek için, [Azure portalı kullanarak Azure rol atamalarını ekleme veya kaldırma](https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal)konusuna bakın.

Azure Active Directory (Azure AD) özel rollerini yönetmek için, [Azure Active Directory 'deki (Önizleme) özel yönetici rollerine](https://docs.microsoft.com/azure/active-directory/users-groups-roles/roles-custom-overview)bakın.

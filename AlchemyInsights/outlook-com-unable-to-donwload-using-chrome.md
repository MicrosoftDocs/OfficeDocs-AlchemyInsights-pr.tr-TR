# <a name="cant-download-files-in-chrome"></a>Chrome 'da dosyaları indiremıyorum

**Sorun** Bazı müşterilerin Google Chrome tarayıcısı kullanırken dosyaları indirebilmelerini engelleyen bir sorunla karşılaştık. 

**Geçici çözüm** [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) veya başka bir Web tarayıcısını kullanın.
[Dosyaları ve Outlook.com eklerini indirme](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy)hakkında daha fazla bilgi edinin.

# <a name="cant-download-files-in-chrome"></a>Krom dosyalar karşıdan yüklenemiyor

**Sorunu** Bazı müşteriler Google Chrome tarayıcı kullanırken dosyaları karşıdan çalışabilme engelleyen bir sorunu haberdar ediyoruz. 

**Geçici çözüm** [Microsoft kenarına](https://www.microsoft.com/en-gb/windows/microsoft-edge) veya başka bir web tarayıcısı kullanın.
[Yükleme dosyalarına](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab)ve Outlook.com ekleri hakkında daha fazla bilgi edinin.

# <a name="block-file-download-with-azure-ad-conditional-access"></a>Azure AD koşullu erişimle dosya indirmeyi engelleme

Bulut App Security oturum ilkeleri, bir oturumu aygıt durumuna göre kısıtlamanıza olanak tanır. Aygıtını koşul olarak kullanarak bir oturumun denetimini gerçekleştirmek için hem koşullu erişim ilkesi hem de oturum ilkesi oluşturun.

- [Azure AD koşullu erişimle Dosya yı engelleme](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

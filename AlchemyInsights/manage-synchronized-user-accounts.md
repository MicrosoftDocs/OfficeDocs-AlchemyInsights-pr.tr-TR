# <a name="manage-synchronized-user-accounts"></a>Eşitlenmiş kullanıcı hesaplarını yönetme

Normalde, kullanıcı hesaplarını eklemek ve değiştirmek için Microsoft 365 yönetim merkezini kullanırsınız. Bununla birlikte, dizinde sychronization veya SSO etkinse bu özelliği kullanaabilirsiniz. Bunun yerine, Kullanıcı hesabı özelliklerini [Office 365 PowerShell](https://docs.microsoft.com/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell )ile yapılandırma konusunda açıklandığı gibi kullanıcılarınızı Windows PowerShell ile yönetin.

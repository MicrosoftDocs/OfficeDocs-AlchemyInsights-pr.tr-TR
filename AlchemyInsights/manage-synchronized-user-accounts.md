# <a name="manage-synchronized-user-accounts"></a>Eşzamanlı kullanıcı hesaplarını yönetme

Genellikle, Office 365 Yönetim Merkezi ekleyebilir ve kullanıcı hesaplarını değiştirmek için kullanın. Ancak, dizin eşitleme veya SSO etkinse, onu kullanamazsınız. Bunun yerine, [Office 365 PowerShell ile kullanıcı hesabı özelliklerini yapılandırma](https://docs.microsoft.com/en-us/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell )bölümünde açıklanan kullanıcılarınızın Windows PowerShell ile yönetin.

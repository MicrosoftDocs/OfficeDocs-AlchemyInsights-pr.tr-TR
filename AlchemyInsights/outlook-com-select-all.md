# <a name="how-to-select-all-in-outlookcom"></a>"Tümünü Outlook.com seçmek" nasıl

Tüm iletileri görünür durumdaysa, yalnızca tüm seçebilirsiniz. Bu sizin göremediğiniz iletileri yanlışlıkla eylemleri önlemek içindir.

Ayrı ayrı birden çok ileti **CTRL + tıklatma** seçer.

**SHIFT + tıklatma** seçtiğiniz ilk ve son ileti arasındaki tüm iletileri seçer.

# <a name="how-to-select-all-in-outlookcom"></a>Outlook.com 'daki "Tümünü Seç"

Tüm iletiler görünür durumdaysa yalnızca tümünü seçebilirsiniz. Bu, göremeyebilirsiniz iletilerinde yanlışlıkla yapılan eylemleri önlemektir.

**CTRL + tıklatma** tek tek mesajları seçer.

**SHIFT + tıklama** seçtiğiniz ilk ve son ileti arasındaki tüm iletileri seçer.

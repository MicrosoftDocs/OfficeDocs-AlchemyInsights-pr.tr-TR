# <a name="how-to-select-all-in-outlookcom"></a>Outlook.com 'de "tümü" nasıl seçilir?

Yalnızca tüm iletiler görünürse tümünü seçebilirsiniz. Bu, göremediğiniz iletilerde yanlışlıkla eylemleri önlemek içindir.

**CTRL + click** birden çok tek ileti seçer.

**SHIFT + click seçtiğiniz** ilk ve son ileti arasındaki tüm iletileri seçer.

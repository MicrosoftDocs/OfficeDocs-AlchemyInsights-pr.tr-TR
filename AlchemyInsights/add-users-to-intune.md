# <a name="add-users-to-intune"></a>Kullanıcıları Intune'a ekleme

 [Microsoft 365 yönetici merkezini](https://admin.microsoft.com/) veya [Intune portalını](https://portal.azure.com/#blade/Microsoft_Intune_DeviceSettings/ExtensionLandingBlade/overview)kullanarak kullanıcıları Intune aboneliğinize el ile ekleyebilirsiniz.

Yönetici, Kullanıcı hesaplarını Intune lisansları atamak için ayarlayabilir. Lisansları Microsoft 365 yönetici merkezine veya Intune portalına atayabilirsiniz.

Microsoft 365 yönetici merkezini kullanma hakkında daha fazla bilgi için [bkz.](https://support.office.com/article/Add-users-individually-or-in-bulk-to-Office-365-Admin-Help-1970f7d6-03b5-442f-b385-5880b9c256ec)

Intune portalını kullanma hakkında daha fazla bilgi için [bkz.](https://docs.microsoft.com/intune/fundamentals/users-add)

# <a name="install-project-professional"></a>Project Professional 'ı yükleme

Project Online 'ın abonelik sürümünü ve Project 2019, 2016 veya 2013 ' nin abonelik dışı sürümünü yüklemeyi öğrenmek için [Project yükleme](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) 'ye bakın. 

Project 2010 veya 2007 için, [office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d)   veya [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657)ile ilgili adımları izleyin. 

Proje tek başına bir uygulama olarak satılır, bu nedenle henüz [Proje satın alın veya deneyin](https://products.office.com/project).

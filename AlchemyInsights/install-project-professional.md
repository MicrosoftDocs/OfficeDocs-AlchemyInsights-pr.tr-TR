# <a name="install-project-professional"></a>Project Professional'i yükleme

Project Online'ın abonelik sürümünü ve Project 2019, 2016 veya 2013 abonelik dışı sürümünün nasıl yük olduğunu öğrenmek için [Project'i](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) Yükleme'ye bakın. 

Project 2010 veya 2007 [için, Office 2010 veya Office 2007'de](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) [adımları izleyin.](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657) 

Project tek başına uygulama olarak satılır, bu nedenle henüz sahip değilken Project'i satın alın [veya deneyin.](https://www.microsoft.com/evalcenter/evaluate-project)

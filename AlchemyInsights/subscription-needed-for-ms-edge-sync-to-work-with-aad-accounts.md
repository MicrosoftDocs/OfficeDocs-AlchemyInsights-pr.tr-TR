# <a name="subscription-needed-for-microsoft-edge-sync-to-work-with-azure-active-directory-accounts"></a>Azure Active Directory hesaplarıyla çalışmak için Microsoft Edge Sync için abonelik gerekiyor

Eşitleme aşağıdaki aboneliklerden herhangi biriyle çalışır:

1. Azure Active Directory Premium (P1 ve P2)
1. Microsoft 365 Iş ekstra
1. Office 365 E3 ve üzeri
1. Azure bilgi koruması (P1 ve P2)
1. Tüm Office 365 Eğitim abonelikleri:
    - Öğrenciler veya Fakülteler için Microsoft uygulamaları
    - Öğrenciler veya Fakülteler için Exchange Online
    - Office 365 a1 veya üstü
    - Microsoft 365 a1 veya üstü
    - Öğrenciler veya Fakülteler için Azure Information Protection P1 veya P2

# <a name="create-users-in-hybrid-deployments"></a>Karma dağıtımlarda kullanıcı oluşturma

Kuruluşunuz şirket içi ortam ve Microsoft 365 arasında karma bir dağıtım olduğunda, şirket içi Etkin Dizininizde tüm kullanıcı nesnelerini oluşturmanız gerekir. Kullanıcı nesnelerini uygun kuruluş biriminde (OU) oluşturursunuz, böylece bunlar kuruluşunuzla senkronize olur.

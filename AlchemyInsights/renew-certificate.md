# <a name="renew-certificate"></a>Sertifikayı Yenile

Yakında sona erecek bir sertifikayı yenilemek için [Azure Active Directory 'de federe çoklu oturum açma için sertifikaları yönetme](https://docs.microsoft.com/azure/active-directory/manage-apps/manage-certificates-for-federated-single-sign-on#renew-a-certificate-that-will-soon-expire)bölümüne bakın.

Belirteç imzalama sertifikalarınızı yönetme ve Azure AD ile eşitlenmiş durumda tutma hakkında ek bilgi için, [Microsoft 365 ve Azure Active Directory için Federasyon sertifikalarını yenileyin](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-fed-o365-certs).

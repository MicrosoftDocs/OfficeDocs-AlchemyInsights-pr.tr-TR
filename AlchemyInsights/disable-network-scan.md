# <a name="disable-network-scan"></a>Ağ taramasını devre dışı bırakma

Ağ paylaşımı taramaları performansı etkileyebilir.  İstemcinin ağ paylaşımlarını/dosyaları varsayılan olarak taramayması için, Windows Defender uygulamasında aşağıdaki ayarları True olarak **yapılandırabilirsiniz:**

- DisableScanningMappedNetworkDrivesForFullScan
- DisableScanningNetworkFiles

# <a name="does-visio-work-on-a-mac"></a>Visio Mac'te çalışıyor mu?

Mac için Visio masaüstü uygulaması yoktur. Ancak, uygulamaya ihtiyacı olan kullanıcının [bir Visio Online lisansı atanmış](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799)olması, bir web tarayıcısında [Visio Online'da diyagramları görüntüleyebilmesi, oluşturabileceği ve edinebileceği](https://support.office.com/article/06f04845-91b8-4e8f-881f-a43c970735fc?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799) sürece.

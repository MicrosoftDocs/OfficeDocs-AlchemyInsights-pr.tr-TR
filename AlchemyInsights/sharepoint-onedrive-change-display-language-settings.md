# <a name="change-the-default-site-language-in-sharepoint-or-onedrive"></a>SharePoint veya OneDrive 'da varsayılan site dilini değiştirme 

**Site oluşturulduktan sonra** SharePoint veya OneDrive için varsayılan site dilini değiştiremezsiniz. Ancak kişisel görüntü dil ayarlarınızı değiştirebilirsiniz.

Kişisel ayarlarınızı değiştirmek için, [kişisel dil ve bölge ayarlarınızı değiştirme](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7)konusuna bakın.

Site Kullanıcı arabirimini değiştirmek için, [sitenin Kullanıcı arabiriminde kullanılabilir olmasını istediğiniz dilleri seçin](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

# <a name="manage-recurring-billing"></a>Yinelenen faturalandırma yönetme

Yinelenen faturalandırma yönetmek için **yinelenen fatura** açmak veya kapatmak geçiş yapabilirsiniz.
  
1. Yönetim Merkezi [abonelikleri](https://go.microsoft.com/fwlink/p/?linkid=842054) sayfasına gidin veya **Faturalama** Git \> **abonelikleri**.

2. Yinelenen faturalandırma yönetmek istediğiniz aboneliği seçin.

3. **Yinelenen faturalandırma** geçiş için **açık** veya **Kapalı**geçin.

Daha fazla bilgi için bkz: [aboneliğiniz için fatura yinelenen Yönet](https://support.office.com/article/8d83b530-f4ca-47f6-a666-e5791cbacc7e).

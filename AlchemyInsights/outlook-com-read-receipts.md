# <a name="read-receipts"></a>Makbuzları okuma

Outlook.com'da okundu bilgisi isteyemezsiniz, ancak Windows için Outlook kullanılarak gönderilen Outlook.com iletileri için okundu bilgisi isteyebilirsiniz. Nasıl yapılacağını öğrenmek için [bkz.](https://support.office.com/article/a34bf70a-4c2c-4461-b2a1-12e4a7a92141?wt.mc_id=Office_Outlook_com_Alchemy)
  
Outlook.com okundu bilgisi isteklerine nasıl yanıt vereceğini seçmek için [Okundu Bilgisi ayarlarını okuyun.](https://outlook.live.com/mail/options/mail/handling/readReceipts)

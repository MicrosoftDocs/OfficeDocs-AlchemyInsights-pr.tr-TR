# <a name="windows-doesnt-start-on-surface"></a>Windows Surface'ta baş değil

Windows Surface cihazınıza başlanmazsa veya cihazınız uyku modundan çıkarmıyorsa şu sorun giderme adımlarını deneyin:

- Surface' ınızı açsanız (Windows başlatmaz) Windows düzgün başlayamezse, sorunu çözme adımları için [Windows'un Surface'ta](https://support.microsoft.com/surface/windows-doesn-t-start-on-surface-3dd47ea1-472a-4514-c8e1-ff81bd72be5c)başlamamanıza bakın.

- Surface'nız aç ve uyanmazsa, sorun giderme adımları için Surface'ın açma veya uykudan [uyanmamalarına bakın.](https://support.microsoft.com/surface/surface-won-t-turn-on-or-wake-from-sleep-1e181652-3db8-5ca1-9649-7390fafb102a)

- Üçüncü taraf bir cihaz kullanıyorsanız, üçüncü taraf [cihazlardaki sorunları gidermeye bakın.](https://support.microsoft.com/topic/b6f3408d-dac9-43e2-82f6-e620ca783636)

# <a name="access-services"></a>Erişim Hizmetleri

SharePoint Online'daki Access Services özelliği kullanımdan kaldırılmaktadır. Yeni Access tabanlı web uygulamaları ve web veritabanları artık oluşturulamaz ve mevcut web uygulamaları ve web veritabanları Nisan 2018'e kadar kapatılır. Access Desktop veritabanları (.accdb) bu değişiklikden etkilenmez. Access web uygulamalarına ve web veritabanlarına alternatif olarak Microsoft PowerApps'ı öneririz. 
  
[Microsoft PowerApps hakkında bilgi edinin](https://powerapps.microsoft.com/)

# <a name="no-subscriptions-found-message-in-the-security-center"></a>Güvenlik Merkezi'nde abonelik bulunamadı iletisi

Microsoft Defender Güvenlik Merkezi'ne erişirken "Abonelik bulunamadı" iletisi alırsanız, portalda kullanıcının oturum a açması için kullanılan Azure Active Directory'nin (AAD) Microsoft Defender ATP lisansı yoktur.  

Windows E5 ve Office E5 lisansları ayrı lisanslardır.

Lisans satın alındı ancak bu AAD örneğine sağlanmazsa bir destek durumu açın. Sahip olduğunuz: <br/>
-   Olası bir lisans sağlama sorunu.<br/>
-   Lisansı, hizmette kimlik doğrulaması için kullanılandan farklı bir Microsoft AAD'ye yanlışlıkla siz sağlamayı kabul ettiysiniz.

# <a name="create-custom-content-types"></a>Özel içerik türleri oluşturma

1. Hızlı Başlatma'da çalışmak istediğiniz listeyi seçin veya **Ayarlar'ı** (vites simgesi) tıklatın ve ardından **Site ayarlarını**tıklatın.
2. **Web Tasarımcısı Galerileri**altında site **içerik** türlerini tıklatın.
3. Site İçerik Türleri sayfasında Oluştur'u tıklatın.
4. Yeni Site İçerik Türü sayfasında, özel içerik türünüzle ilgili bilgileri doldurun.

# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>SharePoint site izinlerini SharePoint Grupları aracılığıyla özelleştirme 

SharePoint grubu, tümü sitelere ve içeriğe aynı izin kümesine sahip kullanıcılar topluluğudur. İzinleri aynı anda bir kişiye atamak yerine, aynı izin düzeyini aynı anda birçok kişiye rahatlıkla atamak için grupları kullanabilirsiniz.

[SharePoint site izinlerini özelleştirme](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)

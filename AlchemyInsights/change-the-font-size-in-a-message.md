# <a name="change-the-font-size-in-a-message"></a>İletide yazı tipi boyutunu değiştirme

1. İletiyi daha önce yazdıysanız metni seçin.
2. İletinin altındaki  **Biçim** simgesini (ilk simge) seçin.
3. İletinin yukarısındaki  **yazı tipi boyutu**  simgesini (yedinci simgesi) seçin.
4. Bir boyut seçin.

1. <span data-ttu-id="e724f-104">Hemen [kullanıcının parolasını sıfırlayın](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c).</span><span class="sxs-lookup"><span data-stu-id="e724f-104">[Reset the user's password](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) immediately.</span></span> <span data-ttu-id="e724f-105">Yeni parolayı son kullanıcıya e-postayla iletmeyin.</span><span class="sxs-lookup"><span data-stu-id="e724f-105">Do not communicate the new password through email to the end user.</span></span>

3. <span data-ttu-id="e724f-107">Posta kutusunun içinde ayarlanmış tüm şüpheli [gelen kutusu kurallarını](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) kaldırın.</span><span class="sxs-lookup"><span data-stu-id="e724f-107">Remove any suspicious [inbox rules](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) set within the mailbox.</span></span> 
    
4. <span data-ttu-id="e724f-108">Kullanıcının e-posta göndermesi engellendiyse, [Kısıtlı Kullanıcılar'a gidip hesabın engelini kaldırın](https://protection.office.com/?hash=/restrictedusers).</span><span class="sxs-lookup"><span data-stu-id="e724f-108">If the user is blocked from sending email, [go to the Restricted Users to unblock the account](https://protection.office.com/?hash=/restrictedusers).</span></span> <span data-ttu-id="e724f-109">Bunu yaptıktan sonra kullanıcının 1 saat içinde ileti göndermeye devam edebilmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="e724f-109">Once done, you should be able to resume sending messages  within 1 hour  .</span></span>

# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Microsoft 365 URL 'Leri ve IP adresi aralıkları

Microsoft 365, *uç noktalar*denen belirli IP adresleri aracılığıyla Internet için bağlantı gerektirir.
Erişilebilir olması gereken uç noktaların listesi için bkz: [Microsoft 365 URL 'leri ve IP adresi aralıkları](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

Microsoft 365 IP adresi ve URL Web hizmeti, değişiklikleri değerlendirmeniz, yapılandırmanız ve güncel kalmanızı kolaylaştırmak için Microsoft 365 ağ trafiğini daha iyi belirleyip ayırt etmenize yardımcı olur. [Microsoft 365 IP adresi ve URL Web hizmeti](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service)'ne bakın.

# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>MacOS üzerinde Microsoft Edge ilke ayarlarını yapılandırmak için Jamf kullanma

macOS üzerinde Microsoft Edge'i (Kararlı kanal 81 veya daha sonraki bir sürümü) yapılandırmak için, Jamf Pro 10.19'da bir Microsoft Edge ilke bildirim dosyası kullanın.

Daha fazla bilgi için [bkz. MacOS üzerinde Jamf ile Microsoft Edge ilke ayarlarını yapılandırma.](https://go.microsoft.com/fwlink/?linkid=2134761)

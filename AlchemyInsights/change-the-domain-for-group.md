# <a name="change-the-domain-for-microsoft-365-group"></a>Microsoft 365 grubu için etki alanını değiştirme

Oluşturma sırasında bir Microsoft 365 grubunun etki alanını değiştirmek istiyorsanız, [microsoft 365 grupları oluştururken kullanılacak etki alanını seçin](https://docs.microsoft.com/microsoft-365/admin/create-groups/choose-domain-to-create-groups)bölümündeki adımları izleyin.

Microsoft 365 grubu zaten oluşturulduktan sonra, etki alanını değiştirme yolu yoktur. Tek yol, yeni bir Microsoft 365 grubu oluşturup gerekirse eskiyi silmelidir (ek e-posta adresleri [Exchange Yönetim merkezinden](https://outlook.office365.com/ecp)eklenebilir.

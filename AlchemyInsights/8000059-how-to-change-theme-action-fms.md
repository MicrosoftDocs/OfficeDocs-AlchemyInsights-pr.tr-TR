# <a name="how-to-change-your-theme"></a>Tema değiştirme

Outlook.com veya Outlook Web tema değiştirmek için **ayarları**' na gidin![ayarları](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) ve **Tema**altında bir tema seçin. Kullanılabilir temaları görmek için **görünümü tüm** **Tema**altında seçin.

# <a name="set-up-spf"></a>SPF'yi ayarlama

Özel etki alanınızdan gönderilen giden e-postayı doğrulamak için özel etki alanınızın yanında bir Gönderen İlkesi Çerçevesi (SPF) kullanabilmeniz için DNS kayıtlarınızı güncelleştirin. Bu, başkalarının postayı etki alanınızdan gönderilmiş gibi "sızdırmalarını" önlemeye yardımcı olur.
  
- [Sızdırmayı önlemeye yardımcı olmak için SPF'yi ayarlama](https://docs.microsoft.com/microsoft-365/security/office-365-security/set-up-spf-in-office-365-to-help-prevent-spoofing)

- [Tüm DNS kayıtlarının ayarlanmasına yardımcı olun](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)

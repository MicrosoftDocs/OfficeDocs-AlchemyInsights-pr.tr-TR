# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>SharePoint Kitaplığını OneDrive Sync ile eşitleme

SharePoint dosyalarınızın yerel sabit diskinizle eşitlemesi için aşağıdaki adımları tamamlayın:

1. Yerel sabit diskinizle eşitlemek istediğiniz belge kitaplığına gidin.
2. Menü seçeneklerinden Eşitle düğmesini seçin.
3. Eşitlemayı tamamlamak için adım adım izleyin.

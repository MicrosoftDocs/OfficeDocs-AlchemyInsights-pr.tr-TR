# <a name="delete-a-communication-site"></a>İletişim site silme

1. Sitesine gidin.
    
2. Sağ üst ayarları simgesini tıklatın.
    
3. **Site ayarları**' nı tıklatın. **Site bilgileri**altında **Bu siteyi sil**' i tıklatın ve sonra **Sil**' i tıklatın.

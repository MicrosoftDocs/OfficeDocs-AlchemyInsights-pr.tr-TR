# <a name="add-an-extension-to-microsoft-edge-from-the-chrome-web-store"></a>Chrome Web mağazasından Microsoft Edge 'e uzantı ekleme

Chrome Web mağazasından Microsoft Edge 'e uzantı eklemek için aşağıdaki adımları izleyin:

1. Chrome Web mağazasına gidin.
2. Sayfanın en üstündeki başlıkta, **diğer mağazalardan uzantılara Izin ver**'i seçin.
3. Eklemek istediğiniz uzantıyı seçin ve ardından **Chrome 'A Ekle 'ye** tıklayın.
4. Onay iletişim kutusunda, uzantının yapabileceğinden değişiklikleri dikkatle gözden geçirin ve sonra **da uzantı Ekle**'yi tıklatın.
Son iletişim kutusu uzantının eklendiğini onaylar.

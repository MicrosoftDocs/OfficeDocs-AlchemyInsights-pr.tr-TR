# <a name="add-an-extension-to-microsoft-edge-from-the-chrome-web-store"></a>Chrome web mağazasından Microsoft Edge'e uzantı ekleme

Chrome web mağazasından Microsoft Edge'e uzantı eklemek için aşağıdaki adımları tamamlayın:

1. Chrome Web Mağazası'ne gidin.
2. Sayfanın en üstünde yer alan başlıkta, diğer mağazalardan **uzantılara izin ver'i seçin.**
3. Eklemek istediğiniz uzantıyı seçin ve ardından **Chrome'a Ekle'ye tıklayın.**
4. Onay iletişim kutusunda, uzantının yapılacak değişiklikleri dikkatle gözden geçirin ve Uzantı **ekle'ye tıklayın.**
Son iletişim kutusu, uzantının ekli olduğunu onaylar.

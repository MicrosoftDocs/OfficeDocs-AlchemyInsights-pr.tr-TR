# <a name="install-and-configure-onedrive"></a>OneDrive 'ı yükleme ve yapılandırma

OneDrive eşitleme istemcisi Windows 10 ile önceden yüklenir. Windows 7 veya Mac için OneDrive 'ı yüklemek veya telefonunuzun veya tabletinizin mobil uygulamasını almak için, şuraya gidin [https://onedrive.com/download](https://onedrive.com/download) .
  
Windows 'da OneDrive eşitleme istemcisini ayarlamak için bkz:
  
- [OneDrive Iş dosyalarınızı eşitlemek için bilgisayarınızı ayarlama](https://go.microsoft.com/fwlink/?linkid=533375)

- [Yeni OneDrive eşitleme istemcisini kullanarak SharePoint dosyalarını eşitleme](https://go.microsoft.com/fwlink/?linkid=871666)

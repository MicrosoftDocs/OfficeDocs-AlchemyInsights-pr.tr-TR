# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Web üzerinde Outlook 'ta hesap devre dışı bırakıldı hatası mu yaşıyorsunuz?

**Your account has been disabled** `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` Web üzerinde Outlook 'u (eski adıyla Outlook Web App) açmaya çalışırken hesabınızın devre dışı bırakıldığı hatayı alırsanız, kuruluşunuzun Yöneticisi Web üzerinde Outlook 'a erişiminizi devre dışı bırakmış olabilir. Yöneticiniz, [bir posta kutusu Için Outlook Web App 'ı etkinleştirme veya devre dışı bırakma](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)konusundaki adımları izleyerek bu hatayı düzeltebilir.

# <a name="add-a-user"></a>Kullanıcı ekleme

1. Microsoft 365 yönetim merkezinde **Kullanıcılar** > [Etkin kullanıcılar](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/users)’a gidin.

2. 
            **Kullanıcı ekleme** seçeneğine tıklayın.

3. Kullanıcı bilgilerini doldurun. Bitirdiğinizde **Eklemeyi sonlandır**’ı seçin.

Daha fazla bilgi edinmek için bkz. [Kullanıcıları tek tek veya toplu olarak ekleme](https://docs.microsoft.com/office365/admin/add-users/add-users).

# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Bir hesap alınırken hata Web'de Outlook'taki devre?

**Hesabınız devre dışı** hata iletisini alırsanız ile `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` (eski Outlook Web App da bilinir) web üzerinde Outlook açmaya çalıştığınızda, kuruluş, yönetim, Outlook Web erişimi devre dışı bırakmış olabilir. Yöneticinizden [etkinleştirmek veya devre dışı bir posta kutusu için Outlook Web App](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)konudaki adımları takip ederek bu hatayı düzeltebilirsiniz.

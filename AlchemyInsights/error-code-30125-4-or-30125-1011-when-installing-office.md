# <a name="error-code-30125-4-or-30125-1011-when-installing-office"></a>Office'i yüklerken 30125-4 veya 30125-1011 hata kodu

Virüsten koruma yazılımınız, güvenlik duvarınız, proxy ayarlarınız veya bağlantınız Office'i yüklemenizi engelliyorsa bu hatayı görebilirsiniz. Sorunu gidermeye çalışabileceğiniz bazı şeyler şunlardır:
  
- Bilgisayarınızı yeniden başlatın ve Office'i yeniden yükleyin

- Office çevrimdışı [yükleyicisini](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA) kullanarak Office'i yükleme

- Proxy ayarlarını geçici olarak kapatın

- Virüsten koruma yazılımLarını geçici olarak kapatın

- Güvenlik duvarını sürme

Daha fazla bilgi için [Office'i yüklerken 30125-4 veya 30125-1011 hata koduna](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612?wt.mc_id=Alchemy_ClientDIA) bakın

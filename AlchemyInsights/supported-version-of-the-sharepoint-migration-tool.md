# <a name="supported-version-of-the-sharepoint-migration-tool"></a>SharePoint Geçiş Aracı'nın desteklenen sürümü



Şu anda SharePoint Geçiş Aracı yalnızca SharePoint 2013'ten gelen geçişleri destekler. 2010 ve 2016 sürümleri şu anda desteklenmez.
  
Daha fazla bilgi için Bkz. [SharePoint Geçiş Aracı'nı tanıtın.](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409)

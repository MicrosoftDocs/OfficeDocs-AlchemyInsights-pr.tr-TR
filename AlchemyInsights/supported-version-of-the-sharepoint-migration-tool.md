# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Desteklenen sürüm SharePoint geçiş aracı

Şu anda SharePoint geçiş aracı, yalnızca SharePoint 2013 geçişi destekler. Sürümleri 2010 ve 2016 şu anda desteklenmiyor.
  
Daha fazla bilgi için bkz: [SharePoint geçiş aracı Tanıtımı](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).

# <a name="view-reports-on-onedrive-activity-and-usage"></a>OneDrive etkinliği ve kullanımıyla ilgili raporları görüntüleme

[Microsoft 365 Yönetim merkezinde kullanım raporları sayfasına](https://admin.microsoft.com/AdminPortal/Home)gidin. **Rapor Seç**'In altında OneDrive etkinliği veya kullanım raporunu seçin.

Daha fazla bilgi için [OneDrive raporları hakkında](https://go.microsoft.com/fwlink/?linkid=875239)konusuna bakın.

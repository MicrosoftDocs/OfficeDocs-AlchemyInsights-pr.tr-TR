# <a name="desktop-app-for-sharepoint"></a>SharePoint için masaüstü uygulaması

Kuruluşunuz, SharePoint için masaüstü uygulamasının özel Önizleme programına girilir.

Microsoft şu anda SharePoint için masaüstü uygulamasını desteklememektedir (özel Önizleme). SharePoint için masaüstü uygulamasıyla ilgili yardım almak için lütfen kayıtlı Microsoft özel önizleme ortaklarınızı kullanın.

# <a name="bulk-delete-user"></a>Toplu silme kullanıcısı

Kullanıcıları toplu olarak silmek için virgülle ayrılmış değerler (CSV) dosyasını kullanarak çok sayıda kullanıcıyı gruptan kaldırabilirsiniz. Azure AD portalLarını kullanarak toplu kullanıcıları silmek için [Azure Etkin Dizini'ndeki toplu silme kullanıcıları](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete)bölümüne bakın.

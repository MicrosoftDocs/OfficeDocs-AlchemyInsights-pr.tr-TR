# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Sibel tanılamak ve Outlook kimlik doğrulama sorunlarını gidermek için kullanın

[Outlook parolamı tamamlayamıyorum](https://aka.ms/SaRA-OutlookPwdPrompt) tanı etkilenen makinede sorunlarını gidermek için Outlook sürekli parola ister burada kullandığınız önerilir. Bu [SaRA](https://diagnostics.office.com/#/) tanı otomatik denetim yapar ve olası çözümleri, algılanan sorunları gidermek için kullanabileceğiniz döndürür.

# <a name="permissions-and-browser-related-requirements-for-microsoft-365-defender"></a>Microsoft 365 Defender için izinler ve tarayıcıyla ilgili gereksinimler

- İzinler: Microsoft 365 Defender'ı açmak için, Azure Active Directory'de genel yönetici veya güvenlik yöneticisi olun. Microsoft 365 Defender'ı kullanabileceğiniz rollerin listesini görmek ve verilere erişimin düzenlemeye tabi olduğunu öğrenmek için [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2143626)erişimini yönetme makalesini kullanın.

- Tarayıcıyla ilgili gereksinimler: Microsoft 365 güvenlik merkezinde Microsoft 365 Defender'a erişmek için Microsoft Edge, Internet Explorer 11 veya HTML 5 uyumlu herhangi bir web tarayıcısı kullanın.

Aşağıdaki kurulum kılavuzlarından birini seçin:

- **Azure Active Directory Connect** [Eşitleme](https://go.microsoft.com/fwlink/?linkid=2071310) kullanıcılarını dizine bağlama Parola karma eşitlemesini veya doğrudan kimlik doğrulamayı yapılandırarak, kullanıcılarının e-postalarında ve şirket içi Active Directory'niz (etki alanı denetleyicisi) için aynı parolayı kullanarak oturum açmalarını s sağlar. Active Directory Federasyon Hizmetleri (AD FS) ile de federasyon oturum açması kurabilirsiniz.

- **Azure özellikleri Azure** AD kurulum [kılavuzu](https://go.microsoft.com/fwlink/?linkid=2134390) Çok faktörlü kimlik doğrulaması, cihaz kaydı ve güvenlik izleme gibi Azure Active Directory Premium'da kullanılabilen gelişmiş özellikleri ayarlama.

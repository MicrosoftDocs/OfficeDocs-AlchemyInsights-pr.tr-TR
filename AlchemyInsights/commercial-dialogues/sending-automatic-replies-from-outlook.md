# <a name="sending-automatic-replies-from-outlook"></a>Outlook'tan otomatik yanıt gönderme

- Sahip olduğu e-posta hesabının türüne bağlı olarak, otomatik iş dışında yanıtları göndermenin iki yolu vardır.
- **Dosya'ya** tıklayın, ardından Outlook **sürümünüze** bağlı olarak Otomatik Yanıtlar'ı seçin.

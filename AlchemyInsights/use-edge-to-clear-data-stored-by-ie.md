# <a name="use-microsoft-edge-to-clear-the-browsing-data-stored-by-internet-explorer"></a>Internet Explorer tarafından depolanan gözatma verilerini temizlemek için Microsoft Edge'i kullanma

Bu özellik yalnızca, kuruluş internet explorer (IE) modunun açık olduğu durumda kullanılabilir. IE gözatma verilerinizi temizlemenin Microsoft Edge gözatma verilerinizi etkilemeyeceğini unutmayın.
*
1. Microsoft Edge'de **Ayarlar'ı ve daha fazlasını seçin...**  >  **Ayarlar**  >  **Gizlilik ve hizmetler**.

1. **Internet Explorer gözatma verilerini temizle'nin altında,** **Nelerin temizleceklerini seçin'i seçin.**

1. Temizlemek istediğiniz veri türlerini seçin.

1. Şimdi **temizle'yi seçin.**

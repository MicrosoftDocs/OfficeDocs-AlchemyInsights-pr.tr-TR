# <a name="remove-an-extension-from-microsoft-edge"></a>Microsoft Edge 'den uzantıyı kaldırma

Microsoft Edge 'den bir uzantıyı kaldırmak için:

1. Microsoft Edge 'i açın.
2. **... > uzantıları**'nı seçin.
3. **Yüklü uzantılar** sayfasında, kaldırmak istediğiniz uzantıyı bulun ve yanındaki **Kaldır** 'ı seçin.
4. Onay iletişim kutusunda **Kaldır**'ı seçin.

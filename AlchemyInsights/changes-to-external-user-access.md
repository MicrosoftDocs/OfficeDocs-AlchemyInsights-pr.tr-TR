# <a name="changes-to-external-user-access"></a>Harici kullanıcı erişimine yapılan değişiklikler

23 Mart 2018 başlayarak dış kullanıcılar artık bunlar davet veya yüklenmiş yetkili gruplara eklenebilir olsa bile, herkes, tüm kimliği doğrulanmış kullanıcılar veya tüm formları kullanıcılar ile paylaşılan içeriğe erişmek mümkün olacaktır. Harici kullanıcılar yalnızca kullanıcı veya kullanıcının ait olduğu grupları ile paylaşılan içeriğe erişmek mümkün olacaktır. Varolan ayarlarınızı korumak istiyorsanız, lütfen bakın:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365

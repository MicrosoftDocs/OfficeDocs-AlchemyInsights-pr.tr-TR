# <a name="add-a-content-type-to-a-list-or-library"></a>Listeye veya kitaplıka içerik türü ekleme

1. [Liste veya kitaplık ayarlarına](https://support.microsoft.com/en-us/office/edit-list-settings-in-sharepoint-online-4d35793b-246e-42a3-990c-563a83795b7f)gidin.
2. **İçerik Türleri**altında, **varolan site içerik türlerinden ekle'yi**seçin. **İçerik Türleri** görünmüyorsa, **Gelişmiş ayarları**seçin.
3. **Listeden Seç Site içerik türlerinden** seçim belirleyin.
4. Kullanılabilir Site İçerik Türleri listesinde, istediğiniz içerik türünün adını seçin ve seçili içerik türünü liste eklemek için İçerik türlerine taşıyın, Ekle'yi seçin.
5. Ek içerik türleri için adımları yineleyin.

Daha fazla bilgi için bkz: Listeye [veya kitaplıkta içerik türü ekle.](https://support.microsoft.com/en-us/office/add-a-content-type-to-a-list-or-library-917366ae-f7a2-47ad-87a5-9689a1884e60)

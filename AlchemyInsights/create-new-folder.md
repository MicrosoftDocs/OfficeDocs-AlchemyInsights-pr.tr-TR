# <a name="create-a-new-folder-in-outlookcom"></a>Outlook.com 'da yeni klasör oluşturma

1. Gezinti Bölmesi 'ndeki **Klasörler** 'In altında **Yeni klasör**'ü seçin.

2. Klasör için bir ad yazıp ENTER tuşuna basın.

Daha fazla bilgi edinmek için [Outlook.com 'da klasör oluşturma](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy)konusuna bakın.

# <a name="error-onedrive-cannot-connect-to-windows"></a>Hata: OneDrive Windows'a bağlanamıyor

Hata iletisini gidermek için, çalışan olup olmadığını görmek için cldflt.sys dosyasını arayın ve çalışmıyorsa etkinleştirin. 
  
CLDflt.sys dosyasını nasıl etkinleştirin, [bkz.](https://go.microsoft.com/fwlink/?Linkid=2031032)

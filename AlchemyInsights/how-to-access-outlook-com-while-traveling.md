# <a name="how-to-access-outlookcom-while-traveling"></a>Seyahat halindeyken Outlook.com 'e erişme

Seyahat halindeyken Outlook.com erişmek için, Microsoft hesabınızın güvenlik bilgilerinizi (güvenlik kodlarını alabileceğiniz bir alternatif e-posta adresi veya telefon numarası) sağlayın. Daha fazla bilgi edinmek için [Outlook.com nasıl erişeceğinizi](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy)öğrenin.

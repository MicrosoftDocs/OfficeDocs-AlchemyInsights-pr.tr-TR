# <a name="how-to-access-outlookcom-while-traveling"></a>Seyahat ederken Outlook.com nasıl erişilir?

Seyahat ederken Outlook.com erişmek için, microsoft hesap güvenlik bilgilerinizin (güvenlik kodları alabileceğiniz alternatif bir e-posta adresi veya telefon numarası) güncel olduğundan emin olun. Daha fazla bilgi edinmek [için seyahat ederken Outlook.com nasıl erişebilirsiniz'](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy)e bakın.

# <a name="issues-while-migrating-data-to-sharepoint-online"></a>SharePoint Online 'a veri geçirilirken karşılaşılan sorunlar

**100TB veri üzerine geçme**

%100 ' den fazla veri SharePoint Online 'a taşınıyor. En kısa zamanda size yardımcı olabilmemiz için aşağıdaki adımları izleyin. 

1. **Yeni hizmet isteği**'ni ve ardından **yeni hizmet isteği**'ni seçin. 
2. Başlık ve açıklamayı **100 TB 'Den SharePoint geçişi**olarak bırakın.
3. Bilet gönderildikten sonra lütfen aşağıdaki bilgilerle güncelleyin: 

    - Geçişin tahmini boyutu.
    - Geçişinizi başlatmak ve tamamlamak istediğiniz zaman tahmin.
    - SharePoint Server, Box, GDrive, dosya paylaşımları gibi içeriği nereden geçirdiğinizi açıklayın.

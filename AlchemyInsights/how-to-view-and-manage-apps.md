# <a name="how-to-view-and-manage-apps"></a>Nasıl görüntüleyeceğinizi ve uygulamaları yönetme

Uygulamalarınızı Microsoft 365 Yönetim merkezinde görüntüleyebilirsiniz. 

1. **Faturalama**için Yönetim Merkezi'nde Git > [ürünleri & hizmetler](https://go.microsoft.com/fwlink/p/?linkid=842054) sayfası.

2. **Ürünleri & Services** sayfasında **uygulamalar** sekmesini seçin.

# <a name="learn-about-bing-maps-licensing"></a>Bing Haritalar lisanslama hakkında bilgi edinin

Bing Haritalar hizmeti, kurum gereksinimlerini ve teknik gereksinimlerini destekleyecek çeşitli esnek lisans seçenekleri sunar. Ayrıca, ücretsiz katmanla Bing Haritalar ile hemen çalışmaya başlayabilirsiniz.

Ayrıntılar için Bing [Haritalar Lisanslama'ya bakın.](https://go.microsoft.com/fwlink/?linkid=2150203)

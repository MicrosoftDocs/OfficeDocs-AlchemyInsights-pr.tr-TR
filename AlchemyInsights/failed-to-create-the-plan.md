# <a name="failed-to-create-the-plan-error-when-trying-to-create-a-planner-tab-in-microsoft-teams"></a>Microsoft Ekiplerde Planner sekmesini oluşturmaya çalışırken "plan oluşturulamadı" hatası

Microsoft Ekiplerde Planner sekmesini oluşturmaya çalışırken "plan oluşturulamadı" hatası alıyorsanız, lütfen aşağıdaki adımları izleyerek yöneticinin grubun üyesi olduğundan emin olun:

1. Yönetim merkezinde, **gruplar**  >  [gruplarına](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/groups)gidin. 
2. Grubu seçin ve **Üyeler** sekmesinde **Tümünü görüntüle ve üyeleri Yönet**'i seçin.
3. Üyeyi ekleyin ve **Kaydet**'i seçin.

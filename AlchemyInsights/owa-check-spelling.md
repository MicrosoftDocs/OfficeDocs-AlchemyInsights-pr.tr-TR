# <a name="use-your-browser-to-check-spelling"></a>Yazımı denetlemek için tarayıcınızı kullanın

Web üzerinde Outlook 'ta yazım denetimi seçeneği yoktur. Yazımınızı denetlemek için Web tarayıcınızı kullanmanız gerekir. Yazım denetimi, Microsoft Edge, Internet Explorer 10 ve sonraki sürümleri ve Firefox, Chrome ve Safari 'nin geçerli sürümleri ile kullanılabilir. Yazım denetimi yapma hakkında daha fazla bilgi edinmek için Web tarayıcınızın seçeneklerini denetleyin.

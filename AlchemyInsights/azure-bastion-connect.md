# <a name="azure-bastion-connect"></a>Azure Bastion Connect

Sanal makinenize (VM) bağlanmak için Azure Bastion kullanmayı deneyin. Sanal makinenize, TLS üzerinden doğrudan Azure portalında güvenli ve sorunsuz RDP bağlantısı sağlar. Azure Bastion aracılığıyla bağlanıyorken, sanal makinenizin genel IP adresine veya RDP trafiğine izin veren gelen NSG kuralına ihtiyacı olmaz. Azure Bastion'un ne olduğu hakkında daha fazla bilgi edinmek için [Azure Bastion | Microsoft Docs](https://docs.microsoft.com/azure/bastion/bastion-overview).

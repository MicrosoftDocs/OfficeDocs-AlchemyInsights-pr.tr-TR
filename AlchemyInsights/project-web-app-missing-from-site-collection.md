# <a name="project-web-app-is-missing-from-the-sharepoint-site-collection"></a>Project Web App SharePoint site koleksiyonunda eksik

Project Web App, Aşağıdaki nedenlerden dolayı SharePoint site koleksiyonunda eksik olabilir:

1. Kuruluşların yalnızca Project Online Essentials lisansı(lar) vardır. Yöneticilerin Project Online Professional ve/veya Premium lisansı eklemesi gerekir. Site bu noktada salt okunur olarak yayınlanacaktır. Daha fazla bilgi için Bkz. [Proje Yönetimi Çözümlerini Karşılaştır.](https://products.office.com/project/compare-microsoft-project-management-software?tab=1)
2. Kiracı, Project Web App Site sağlamanın geciktiği bir sağlama sorunuyla karşılaştı. 24 saatten fazla bir süre geçtiyse ve sorunu yaşamaya devam ediyorsanız, lütfen bir destek bileti günlüğe kaydedin. 24 saatin uzun bir zaman gibi görünebileceğini biliyoruz. Çoğu durumda, zaten bir çözüm üzerinde çalışıyoruz. Lütfen bir çözümü tamamlamamız için bize en az 24 saat verin.

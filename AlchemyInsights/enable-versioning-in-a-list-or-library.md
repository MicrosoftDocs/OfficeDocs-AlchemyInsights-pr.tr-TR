# <a name="enable-versioning-for-a-list-or-library"></a>Liste veya kitaplık için sürüm oluşturmayı etkinleştirme

1. Listeye veya kitaplığa gidin.
    
2. **Ayarlar**'ı ve ardından **kitaplık ayarları** 'nı veya **Liste ayarları**'nı tıklatın.
    
3. **Sürüm oluşturma ayarları**'nı tıklatın.
    
4. İstediğiniz ayarları seçin ve **Tamam**'ı tıklatın.

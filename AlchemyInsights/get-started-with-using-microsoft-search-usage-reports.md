# <a name="get-started-with-using-microsoft-search-usage-reports"></a>Microsoft Arama kullanım raporlarını kullanmaya başlama

Microsoft Arama kullanım raporları, aramanın kurumda nasıl işlevli olduğuyla ilgili içgörüler sağlar. Bu içgörüler, içeriği bu şekilde bulmanıza ve aramanın, kuruluşlarınız kullanıcıları için daha kullanışlı ve kullanışlı bir deneyime neden olur.

Daha fazla bilgi edinmek için [Microsoft Arama Kullanım Raporları'ne bakın.](https://go.microsoft.com/fwlink/?linkid=2152048)

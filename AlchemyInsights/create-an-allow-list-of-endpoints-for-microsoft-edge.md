# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>Microsoft Edge için uç noktaları listesi oluşturma

Microsoft Edge özelliklerinin elden çıkarılmasında tam olarak sahip olması için, bu sunucunun Internet 'e bağlı olması gerekir. Ayrıca, tarayıcının güvenlik duvarları ve Internet 'teki diğer güvenlik mekanizmaları aracılığıyla iletişim kurmasına olanak tanımak için, [Izin verilen uç noktaları listesine](https://go.microsoft.com/fwlink/?linkid=2135054)bir etki alanı URL 'si eklemeniz gerekir.

# <a name="proxyaddress-incorrect"></a>ProxyAddress yanlış

Bir nesne Azure AD ile eşitlendiğinde, Active Directory 'deki proxyAddresses özniteliğinde belirtilen değerler Azure AD kurallarıyla karşılaştırılır ve ardından proxyAddresses özniteliği Azure AD 'de doldurulur. Bu nedenle, Active Directory 'deki nesne proxyAddresses özniteliğinin değerleri, Azure AD 'deki proxyAddresses özniteliğinin değerleriyle aynı olmayabilir.
  
ProxyAddress 'in nasıl doldurulacağından ilgili daha fazla bilgi edinmek için, [Azure AD 'de ProxyAddress özniteliğinin nasıl doldurulduğuna](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad)bakın.

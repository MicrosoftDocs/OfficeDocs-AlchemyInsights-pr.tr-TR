# <a name="proxyaddress-incorrect"></a>ProxyAddress yanlış

Bir nesne Azure AD ile eşitlendiğinde, Etkin Dizin'deki proxyAdres özniteliğinde belirtilen değerler Azure AD kurallarıyla karşılaştırılır ve proxyAdres özniteliği Azure AD'de doldurulur. Bu nedenle, Etkin Dizin'deki nesne için proxyAdres özniteliğinin değerleri, Azure AD'deki proxyAdres özniteliğinin değerleriyle aynı olmayabilir.
  
Proxy adresinin nasıl doldurulur hakkında daha fazla bilgi edinmek için [proxyAddress özniteliğinin Azure AD'de nasıl doldurulur'a](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad)bakın.

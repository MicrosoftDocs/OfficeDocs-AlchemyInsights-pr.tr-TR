# <a name="view-the-management-status-of-microsoft-apps-by-using-microsoft-edge-for-ios-or-android-devices"></a>İOS veya Android cihazlar için Microsoft Edge kullanarak Microsoft uygulamalarının yönetim durumunu görüntüleme

Yönetilen iOS veya Android uygulamalarınızda sorun gidermeye yönelik uygulama günlüklerine erişmek için, Microsoft Edge 'i açın ve Adres kutusuna **hakkında: intunehelp** yazın. Microsoft Edge, sorun giderme modunda başlatılır.

Uygulama günlüklerinde depolanan ayarların listesi için bkz: [istemci uygulaması koruma günlüklerini gözden geçirme](https://docs.microsoft.com/mem/intune/apps/app-protection-policy-settings-log).

Android cihazlarda uygulama günlüklerinin nasıl görüntüleneceğini öğrenmek için [karşıya yükleme ve e-posta günlükleri](https://docs.microsoft.com/mem/intune/user-help/send-logs-to-your-it-admin-by-email-android)bölümüne bakın.

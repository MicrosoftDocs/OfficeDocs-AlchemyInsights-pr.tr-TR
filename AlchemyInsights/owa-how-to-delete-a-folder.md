# <a name="how-to-delete-a-folder-in-outlook-on-the-web"></a>Web üzerinde Outlook 'ta klasör silme

1. Klasör bölmesinde, silmek istediğiniz klasöre veya alt klasöre sağ tıklayın.
2. **Klasörü Sil**'i seçin.
3. Onaylamak için **Tamam 'ı** seçin.

Daha fazla bilgi edinmek için [Web üzerinde Outlook 'ta ileti klasörleriyle çalışma](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f)konusuna bakın.

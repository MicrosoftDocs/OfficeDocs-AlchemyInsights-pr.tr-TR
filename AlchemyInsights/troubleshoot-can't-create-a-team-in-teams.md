# <a name="what-to-do-if-you-cant-create-a-team-in-microsoft-teams"></a>Microsoft ekiplerde ekip oluşturamadığınızda ne yapmalısınız?

Ekipinizden aşağıdakileri yapmasını isteyin:

1. Microsoft 365 uygulamasında kuruluşun güvenlik grubu ayarlarını oluşturun veya değiştirin.
2. Bu güvenlik grubuna ekleyin.

# <a name="solutions-for-unsupported-operating-system"></a>"Desteklenmeyen"işletim sistemi için çözümler



Office 365'i yüklemeye çalıştığınızda, aygıtınız için [sistem gereksinimleri](https://products.office.com/office-system-requirements)karşılamıyorsa, **desteklenmeyen işletim sistemi** hata iletisi görebilirsiniz.
  
Office 365 gerektirir:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 veya Windows Server 2008 R2
    
    VEYA
    
- Mac OS X 10.10
    
Taşınabilir aygıt ya da bir Chromebook Office yüklemeye çalışırken bu ileti aldım, [Microsoft Office bir Chromebook üzerinde çalışması ve yüklenmesi nasıl](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA)bakın.

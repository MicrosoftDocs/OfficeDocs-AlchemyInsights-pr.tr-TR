# <a name="solutions-for-unsupported-operating-system"></a>"Desteklenmeyen işletim sistemi" çözümleri

Microsoft 365 'i yüklemeye çalıştığınızda, cihazınız [sistem gereksinimlerini](https://products.office.com/office-system-requirements)karşılamıyorsa desteklenmeyen bir **işletim sistemi** hata iletisi görebilirsiniz.
  
Microsoft 365 şunları gerektirir:
  
- PC: Windows 10, Windows 8,1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 veya Windows Server 2008 R2

    VEYA

- Mac OS X 10,10

Bu iletiye Office 'i mobil cihaza veya bir Kmebook 'a yüklemeye çalışırken, [bir Kmebook 'A Microsoft Office 'i yükleme ve çalıştırmayı öğrenin](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).

"Desteklenmeyen işletim sistemi"

# <a name="solutions-for-unsupported-operating-system"></a>"Desteklenmeyen"işletim sistemi için çözümler

Office 365/2016 yüklemeye çalıştığınızda, aygıtınız için [sistem gereksinimleri](https://products.office.com/office-system-requirements)karşılamıyorsa, aşağıdaki hata iletisini görebilirsiniz.
  
 **"Desteklenmeyen işletim sistemi"**
  
Office 365/2016 gerektirir:
  
PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 veya Windows Server 2008 R2
  
VEYA
  
Mac OS X 10.10
  
Taşınabilir aygıt ya da bir Chromebook Office yüklemeye çalışırken bu ileti aldım, [Office uygulamaları ve mobil aygıtta e-posta ayarlama](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx)konusuna bakın.

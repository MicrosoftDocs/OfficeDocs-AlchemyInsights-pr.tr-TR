# <a name="recover-missing-or-deleted-site-collections"></a>Eksik veya silinmiş site koleksiyonlarını kurtarma

SharePoint 'te eksik veya silinmiş site koleksiyonlarını kurtarma hakkında bilgi edinmek için aşağıdaki makalelere bakın.

[Silinen siteleri geri yükleme](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[SPO PowerShell modülünü kullanarak geri yükleme](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

Siteyi bulamıyorsanız, [Microsoft 365 güvenlik &amp; Uyumluluk Merkezi 'nde denetim günlüğünde arama](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance)yapmak isteyebilirsiniz.

**Not:** "Modern" ekip siteleri **Klasik** SharePoint Yöneticisi Kullanıcı arabiriminde görünmez. [Yeni SharePoint Yönetim merkezinden](https://docs.microsoft.com/sharepoint/get-started-new-admin-center)"modern" sitelerin listesine erişebilirsiniz.

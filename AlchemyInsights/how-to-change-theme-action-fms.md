# <a name="how-to-change-your-theme"></a>Temanızı değiştirme

Outlook.com veya Web üzerinde Outlook 'ta temanızı değiştirmek için **Ayarlar** 'a gidin ve **Tema**'nın altında bir tema seçin. Kullanılabilir tüm temaları görmek için **Tema**altında **Tümünü görüntüle** 'yi seçin.

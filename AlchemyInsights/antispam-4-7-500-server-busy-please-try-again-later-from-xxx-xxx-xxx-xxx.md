# <a name="47500-server-busy-please-try-again-later"></a>4.7.500 Sunucu meşgul, daha sonra tekrar deneyin

Bu hata, kaynak IP adresinden gelen e-posta trafiğinin hacmi, kaynak IP adresinin itibarına (veya itibar eksikliğine) bağlı olarak sınırı aştığında oluşur.

Kaynak IP adresinden gelen e-postanın engellenmesi bir saat içinde sona erer. Kaynak IP adresi size ait şirket içi bir e-posta sunucusuysa, posta akışı bağlayıcısının yapılandırmasını doğrulayın. Davranış bir saatten uzun süre devam ederse, kaynak IP adresi için bir özel durum istemek için desteğe başvurun.

# <a name="form-header---web-resource"></a>Form üstbilgi - web kaynağı

Formu kullanıcılar için daha çekici veya kullanışlı hale getirmek için bir formdaki web kaynaklarını ekleyebilir veya edebilirsiniz. Başlamak için, Sistem Yöneticisi veya Sistem Özelleştirici güvenlik rolüne veya eşdeğerizinlere sahip olduğundan emin olun.  

**Not** Bir web kaynağını form üstbilgiveya altbilgine dahil edemezsiniz.

Daha fazla bilgi için [bkz.](https://docs.microsoft.com/dynamics365/customer-engagement/customize/create-edit-web-resources#create-and-edit-a-web-resource-on-a-form)

# <a name="how-to-enable-seamless-sso"></a>Kesintisiz SSO 'yu etkinleştirme

[Azure AD Connect](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect)aracılığıyla kesintisiz SSO 'yu etkinleştirin.
  
Azure AD Connect 'in yeni yüklemesini yapıyorsanız, [özel yükleme yolunu](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-get-started-custom)seçin. **Kullanıcı oturum açma** sayfasında **Tekli oturum açmayı etkinleştir** seçeneğini belirleyin.
  
Kesintisiz SSO 'yu doğru şekilde etkinleştirdiğinizden emin olmak için:
  
1. [Azure Active Directory Yönetim Merkezi](https://aad.portal.azure.com) 'nde genel yönetici olarak oturum açın.

2. Sol bölmedeki **Azure Active Directory** 'yi seçin.

3. Kesintisiz çoklu oturum açma özelliğinin **etkinleştirildiğini**doğrulayın.

Daha fazla bilgi edinmek için, bkz [: Azure Active Directory kesintisiz çoklu oturum açma: hızlı başlangıç](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start).

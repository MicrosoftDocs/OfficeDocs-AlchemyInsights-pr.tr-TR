# <a name="web-only-limited-access"></a>Sınırlı erişim Web-Only

Web-Only sınırlı erişim, kullanıcıların yalnızca Web nesnesine erişimini sağlayan ' kısıtlı erişim ' izin düzeyinin bir türevi.

Daha fazla bilgi için bkz [.](https://docs.microsoft.com/sharepoint/understanding-permission-levels#default-permission-levels)

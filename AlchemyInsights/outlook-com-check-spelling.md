# <a name="use-your-browser-to-check-spelling"></a>Yazım denetimi için tarayıcınızı kullanın

Outlook.com bir yazım denetimi işlevi gerekmese de, çoğu web tarayıcısı otomatik olarak Microsoft Edge, Internet Explorer 10 ve sonraki sürümleri ve geçerli sürümleri Firefox, Chrome ve Safari dahil olmak üzere, Yazımınızı denetleyin. Yazım denetleme hakkında daha fazla bilgi, web tarayıcınızın ayarlarını denetleyin.

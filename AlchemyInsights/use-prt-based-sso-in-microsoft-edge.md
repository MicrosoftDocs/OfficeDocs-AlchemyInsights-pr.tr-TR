# <a name="use-prt-based-sso-in-microsoft-edge"></a>Microsoft Edge'de PRT tabanlı SSO kullanma

Birincil [Yenileme Belirteci (PRT),](https://go.microsoft.com/fwlink/?linkid=2133632) Windows 10, iOS ve Android cihazlarında kimlik doğrulaması için kullanılan bir Azure Active Directory anahtarıdır. PRT, bu cihazlarda kullanılan tüm uygulamalar için SSO'ya olanak tanır.

Microsoft Edge'de PRT tabanlı SSO desteği yereldir ve ek uzantıya gerek yoktur. Windows 10 RS3 ve üzerini kullanan oturum açık kullanıcılar, PRT tabanlı SSO desteği olan web sitelerine PRT ile SSO'ya sahip olur.

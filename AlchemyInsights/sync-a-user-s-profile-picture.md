# <a name="sync-a-users-profile-picture"></a>Kullanıcının profil resmi eşitleme

SharePoint çevrimiçi profil resmi eşitleme fotoğrafını eşitleme başlatmak için kullanıcının eylemi gerektirir. Otomatik eşitleme SharePoint Online fotoğraf yoktur. SharePoint çevrimiçi yalnızca fotoğrafları çevrimiçi Exchange, Active Directory Azure eşitler.
  
Kullanıcı hiçbir zaman eşitleme belirtmeyi tercih bir eylem gerçekleştirdi ya da kendi profil resmi ve yeni bir SharePoint çevrimiçi görünmüyor güncelleştirilmiş kullanıcı eşitleme için sor gerekebilir. İşte nasıl bunu yapabilirsiniz:
  
1. Bir özel inç veya Incognito oturumu ve [Office.com'a gitmek](https://na01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.office.com%2F&amp;data=02%7C01%7C%7Cbad62c504a36446096e108d614dec653%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C636719344369977864&amp;sdata=Km7ZnN8FHSouZbxOiEpQAGIKsK82SHr25uYCh3Gc%2F3s%3D&amp;reserved=0)bir web tarayıcısı açın.
  
2. İş veya Okul hesapla oturum açın ve OneDrive veya SharePoint Döşe'yi tıklatın.
  
[Office 365'için profil fotoğrafınızı ekleyin](https://go.microsoft.com/fwlink/?linkid=875585)
  
[Office 365'te profil resmi eşitleme hakkında bilgi](https://go.microsoft.com/fwlink/?linkid=875586)

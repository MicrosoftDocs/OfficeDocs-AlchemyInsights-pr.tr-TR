# <a name="change-my-profile-picture-or-name-in-outlookcom"></a>Outlook.com'da profil resmimi veya adımı değiştirme

1. Microsoft [hesap bilgilerinize](https://go.microsoft.com/fwlink/p/?linkid=860841)gidin.
1. **Adı Edit** veya **Resmi Değiştir'i**seçin.

Lütfen profil fotoğrafınızda veya adınızdaki değişikliklerin yürürlüğe girmesi için 24 saate kadar bekleyin.

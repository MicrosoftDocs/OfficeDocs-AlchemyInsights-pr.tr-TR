# <a name="change-language"></a>Dili Değiştir

1.    Outlook web üzerinde oturum açın.
2. Sayfanın üstündeki **ayarları**seçin > **tüm Outlook ayarlarını görüntüleyin**.
3. **Genel**seçin > **Dil ve saat**.
4. Kullanmak istediğiniz dili seçin.
5. **Kaydet** 'i seçin.
 
Daha fazla bilgi için bkz: [saat dilimi ve dil ayarlarını Outlook web üzerinde değiştirme](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273).

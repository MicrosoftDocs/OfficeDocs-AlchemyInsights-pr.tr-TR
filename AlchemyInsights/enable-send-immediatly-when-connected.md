# <a name="enable-send-immediately-when-connected"></a>Bağlandığında hemen göndermeyi etkinleştir
 
1. Dosya sekmesinde **Seçenekler**'e tıklayın.

2. Outlook seçenekleri iletişim kutusunda **Gelişmiş**'i tıklatın.

3. Gönder ve Al bölümünde, **bağlantı kurulduğunda hemen göndermeyi**etkinleştirmek için öğesine tıklayın. **Tamam**'a tıklayın.

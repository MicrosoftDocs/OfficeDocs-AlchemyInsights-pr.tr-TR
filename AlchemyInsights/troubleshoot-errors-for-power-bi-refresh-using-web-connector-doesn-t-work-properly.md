# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Web Connector düzgün çalışmıyor kullanarak Yenile

[Web.Page](https://msdn.microsoft.com/library/mt260924.aspx) işlevini kullanarak bir web connector komut dosyası varsa ve 18 Kasım 2016 sonra dataset ya da rapor güncelleştirdikten düzgün çalışması için sırayla yenileme için bir ağ geçidi kullanmanız gerekir.

# <a name="submit-microsoft-cloud-for-healthcare-support-requests"></a>Sağlık destek istekleri için Microsoft Cloud 'ı gönderme

Sağlık için Microsoft bulut müşteri lütfen yeni [Power Platform Yönetim Merkezi](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow) aracılığıyla aramanızı gönderin.

Yeni Portal, sağlık için Microsoft bulut için en iyi duruma getirilmiş yardımlı destek için daha fazla yol sağlar.

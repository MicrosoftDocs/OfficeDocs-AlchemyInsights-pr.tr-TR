# <a name="set-passwords-to-never-expire"></a>Set parolaların süresinin hiçbir zaman sona

TÜM parolaların hiçbir zaman süresi dolacak şekilde ayarlamak için aşağıdaki adrese gidin [Güvenlik &amp; gizlilik](https://portal.office.com/adminportal/home#/settings/security). **Parola İlkesi**yanındaki **Düzenle** ' yi tıklatın ve kaydırıcıyı **üzerinde**açın.
  
Daha fazla bilgi için bkz: [, kuruluşunuz için parola süre sonu ilkesi ayarlayın.](https://support.office.com/article/0f54736f-eb22-414c-8273-498a0918678f)

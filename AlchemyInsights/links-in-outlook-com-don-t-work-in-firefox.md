# <a name="links-in-outlookcom-dont-work-in-firefox"></a>Outlook.com'daki bağlantılar Firefox'ta çalışmıyor

Outlook.com e-posta iletisindeki bağlantılar Firefox tarayıcılarında görüntülendiğinde çalışmaz. Mozilla Firefox için [bir düzeltme üzerinde çalışıyor,](https://go.microsoft.com/fwlink/p/?linkid=2001502&amp;clcid=0x409) ama o zamana kadar:
  
- Lütfen [Microsoft Edge](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409) veya başka bir web tarayıcısı kullanın.
- Okuma bölmesinde e-postayı görüntüleyin. Daha fazla bilgi [için, Outlook.com posta kutunuzun görünümünü nasıl değiştireceğinizhakkında](https://support.office.com/article/b41c2ecb-f23c-42b3-b7f8-659646d5e58c?wt.mc_id=Office_Outlook_com_Alchemy)bilgi edinin.

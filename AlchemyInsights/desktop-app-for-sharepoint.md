# <a name="desktop-app-for-sharepoint"></a>SharePoint için masaüstü uygulaması

Kuruluşunuz SharePoint için Masaüstü Uygulaması için Özel Önizleme programına girilir. Microsoft şu anda SharePoint için Masaüstü Uygulamasını (Özel Önizleme) desteklememektedir. SharePoint için Masaüstü Uygulaması ile ilgili yardım almak için lütfen kayıtlı Microsoft Özel Önizleme iş ortaklarınızı kullanın.

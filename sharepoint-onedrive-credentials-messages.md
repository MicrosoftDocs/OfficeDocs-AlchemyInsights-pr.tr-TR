# <a name="credential-messages-in-sharepoint-or-onedrive"></a>Kimlik bilgisi iletilerindeki SharePoint veya OneDrive

<p>Kullanıcıların düzenli olarak Office tarafından SharePoint çevrimiçi veya OneDrive için kimlik belgeleri açmak çalışırken istenebilir.</p> <p>Kimlik bilgisi iletilerin bazı örnekler şunlardır:</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Kuruluş hesabınız ile oturum açın</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Kimlik bilgileri gerekli</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Şu anda bağlanma, ağ bağlantınızı denetleyin bağlanamıyoruz.</p> <p>Bu belirtilerle karşılaşıyorsanız, lütfen <a href="https://support.microsoft.com/en-us/help/2913639/office-applications-periodically-prompt-for-credentials-to-sharepoint">Bu makalede</a>yer alan adımları deneyin.</p> <p>Kimlik doğrulaması için çoklu oturum açma (SSO) kullanıyorsanız, alternatif olarak, lütfen <a href="https://support.microsoft.com/en-us/help/4025962/cant-sign-in-after-update-to-office-2016-build-16-0-7967-on-windows-10">Bu makalede</a> bunun yerine bakın.</p> <p>&nbsp;</p>
